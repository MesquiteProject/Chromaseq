Chromaseq needs to know what to name sequences that are produced when chromatograms are processed. To do this, you will need to specify how to name sequences. Currently, there are only two options: either they will simply be given the sample code as a name, or they will be given the names that are stored in a text file that provides a translation between a sample code and a sequence name.

Sample Codes and Sequence Names File

If you wish to have the sample codes that are contained in the chromatogram file names translated into other names (for example, the full names of the species or samples), you will need to provide a sample codes & names file. There are two formats that can be used for this file, the newer, tab-delimited version, and the "classic" version.

Tab-delimited format

The recommended tab-delimited format for this file is as follows:

code<tab>name for file names<tab>title for name#1<tab>title for name#2<tab>title for name#3<tab>title for name#4
<sample code><tab><name for file names><tab><your name#1><tab><your name#2><tab><your name#3><tab><your name#4>
<sample code><tab><name for file names><tab><your name#1><tab><your name#2><tab><your name#3><tab><your name#4>
<sample code><tab><name for file names><tab><your name#1><tab><your name#2><tab><your name#3><tab><your name#4>
<sample code><tab><name for file names><tab><your name#1><tab><your name#2><tab><your name#3><tab><your name#4>

The first line must contain the titles for the columns. The first column must contain the sample code. The second column must contain the names you wish for any file names to be called for those sequences. These file names should be short, and not contain any odd characters (such as "/") that might confuse the operating system of your computer. Third and later columns contain the names that the sequences are to be called. You will get a choice as to which column will be used to name the sequences.

Here is part of an example file:

code file name long name long name with locality
1120 Bemb perspicuum AZ 1120 Bembidion perspicuum AZ 1120 Bembidion perspicuum AZ: Charleston Bridge 1120
1173 Bemb antiquum ON 1173 Bembidion antiquum ON 1173 Bembidion antiquum ON: Zimmerman 1173
1250 Bemb louisella NS 1250 Bembidion louisella NS 1250 Bembidion louisella NS: Cape North 1250
1514 Bemb antiquum NC 1514 Bembidion antiquum NC 1514 Bembidion antiquum NC: Huntdale 1514
1598 Bemb rufotinctum NC 1598 Bembidion rufotinctum NC 1598 Bembidion rufotinctum NC: Linville River 1598
1626 Bemb antiquum IA 1626 Bembidion antiquum IA 1626 Bembidion antiquum IA: Decorah 1626
1856 Bemb paraenulum VA 1856 Bembidion paraenulum VA 1856 Bembidion paraenulum VA: Big Otter River 1856
1960 Bemb rothfelsi MO 1960 Bembidion rothfelsi MO 1960 Bembidion rothfelsi MO: Big River 1960
2320 Bemb perspicuum CA 2320 Bembidion perspicuum CA 2320 Bembidion perspicuum CA: Pine Creek 2320

 

Version 1.0 ("classic") format

The format of the sample codes & names file is:

<sample code><tab><short sample name><tab><long sample name>;
<sample code><tab><short sample name><tab><long sample name>;
<sample code><tab><short sample name><tab><long sample name>;
<sample code><tab><short sample name><tab><long sample name>;

The sample code is the code present in the chromatogram file name. The short sample name is the name to which this will be translated in some places in the Phred/Phrap/Mesquite system. The long sample name is the name to which it will be translated to in the sequence names of process sequences. The long name is optional. The short name is used for file names, to ensure that file names are not longer the maximum file name length of the operation system.

Here is part of an example file:

1120<tab>Bemb perspicuum AZ 1120<tab>Bembidion perspicuum AZ 1120 ;
1173<tab>Bemb antiquum ON 1173<tab>Bembidion antiquum ON 1173 ;
1250<tab>Bemb louisella NS 1250<tab>Bembidion louisella NS 1250 ;
1514<tab>Bemb antiquum NC 1514<tab>Bembidion antiquum NC 1514 ;
1598<tab>Bemb rufotinctum NC 1598<tab>Bembidion rufotinctum NC1598 ;
1626<tab>Bemb antiquum IA 1626<tab>Bembidion antiquum IA 1626 ;
1856<tab>Bemb paraenulum VA 1856<tab>Bembidion paraenulum VA 1856 ;
1960<tab>Bemb rothfelsi MO 1960<tab>Bembidion rothfelsi MO 1960 ;
2320<tab>Bemb perspicuum CA 2320<tab>Bembidion perspicuum CA 2320 ;

Sample codes and sequence names are not allowed to start with a ".", and they are not allowed to contain any semicolons.

An XML format is being developed for this information, but it is not yet fully supported.
