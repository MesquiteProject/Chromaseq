Segregating Chromatograms

There are two features for segregating chromatograms. The first, and simplest, is Segregate Chromatograms of Listed Codes, available in the Utilities menu when the Mesquite Log window is frontmost. This will segregate into a separate directory all chromatograms whose sample code is included in a text file listing sample codes.

For example, for the following directory:

And imagine you supplied a text file that contained only:

1250
2320

Choosing Segregate Chromatograms of Listed Codes will then cause Mesquite to ask you to choose a File Naming Rule:

If your file naming rule specifies that the sample codes are the text after "DNA" and before the next "_", then Segregate Chromatograms of Listed Codes would segregate those six files that contain either "DNA1250_" or "DNA2320_" in their file names.

The more complex, slower version is Segregate Matching Chromatograms. This will move all of the chromatogram files whose sequence names or gene fragment name (as determined using the same translation file used elsewhere by Chromaseq) contain a particular piece of text.

If the Segregate Chromatogram feature was used specifying that the sequence name should contain "louisella":

Then all chromatograms whose sequence name contains "louisella" would be moved into a directory called "louisella":

Similarly, if you asked to segregate all of the chromatograms that are for a gene fragment whose name contains "28S":

then all of the chromatograms that match that criterion will be segregated into a subdirectory called "28S":

 

You may wish to build a special-purpose Sample Codes & Sequence Names File for use with Segregate Chromatograms, that contains in the long sequence name field a fuller taxonomic hierarchy of the sequence, so that you may search for those other taxonomic groups when using the feature. For example, this Sample Codes & Sequence Names file contains a more complete taxonomic hierarchy in the long sequence name:

1120Bembidion perspicuum AZ 1120Coleoptera Adephaga Carabidae Trechitae Bembidiina Bembidion (Ocydromus) perspicuum ;
1780Hydroscapha natans AZ1780 Coleoptera Myxophaga Hydroscaphidae Hydroscapha natans ;
1812Cyphocoleus miricollis 1812Coleoptera Adephaga Carabidae Harpalinae Odocanthini Cyphocoleus miricollis Fauvel ;

If the text used to segregate chromosomes was then the family name "Carabidae", then all sequences for carabids would be segregated.
