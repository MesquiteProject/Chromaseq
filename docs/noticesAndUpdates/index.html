Chromaseq    

Chromaseq: a package for processing chromatograms and sequence data in Mesquite

Chromaseq is a package in Mesquite that processes chromatograms, makes contigs, base calls, etc., using in part the programs Phred and Phrap. For more information on Chromaseq, including screen shots and an overview, see the Chromaseq Manual. You can also download it from there.
