In the chromatogram viewer, you can use the arrow keys to quickly jump to certain features of the chromatogram. These features can be toggled on/off from the Chromatograms > Arrow Keys menu.

  • Move To Next Base: Using the left or right arrow keys will move the cursor one base to the left or right, respectively
  • Jump To Next Source Read Conflicts with Higher Quality Read: Jumps to the next site where the phred/phrap processed sequence conflicts with a high quality base call in one or more chromatograms
  • Jump To Next Read-Matrix Conflict: Moves cursor to next site where the base call in the data matrix differs from the base call in the chromatogram(s). Because most chromatograms do not call ambiguities (except for 'N'), bases which phred/phrap/Mesquite have called as ambiguous (R, Y, M, K, S, W, H, B, V, D, N) will usually conflict with chromatograms.
  • Jump To Next Read-Read Conflict: Jumps to next site where chromatogram base calls differ
  • Jump To Next Low Quality: Moves cursor to next site where phred/phrap quality is below a certain threshold (see below for threshold settings)
  • Jump To Next Ambiguity: Jumps to next site at which phred/phrap/Mesquite have made an ambiguous base call

You can set the thresholds for conflicts and low quality from the same submenu (Chromatograms > Arrow Keys):

  • Jump Conflict Threshold…: Sets the threshold for conflicts between reads
  • Jump Quality Threshold…: Sets the threshold for quality scores (when Jump to Next Low Quality is toggled on, the cursor will move to the next site with a quality score less than the threshold)
